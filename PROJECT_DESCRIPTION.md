# RENALGUARD AI - Project Description

## Short Description

**RENALGUARD AI** is an AI-powered clinical decision support system designed for primary care physicians to manage chronic kidney disease (CKD) patients. It combines real-time patient monitoring, evidence-based risk assessment (KDIGO 2024), and AI-driven treatment recommendations to help doctors identify kidney disease early, track progression accurately, and optimize treatment strategies. The platform features an intelligent AI Doctor Assistant, automated lab analysis with smart alerting, and comprehensive patient management tools - all built with React, TypeScript, Node.js, PostgreSQL, and Anthropic's Claude AI.

## Tech Stack

- **Frontend**: React 19 + TypeScript + Vite + Tailwind CSS
- **Backend**: Node.js + Express + PostgreSQL
- **AI**: Claude Sonnet 4.5 by Anthropic
